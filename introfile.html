The Pre-formatted tag

When we write some content in the paragraph tag, the mistakes like extra spaces or unnecessary enters get auto corrected.
But if we use the 'pre' tag, we see the text as it is

For example, if we write :- "Hi I am Asit. My              age is      18.

I study in IIT Hyderabad" without using pre tag-

We get - Hi I am Asit. My age is 18. I study in IIT Hyderabad

The blockquote tag indents the text in the webpage.
The 'q' tag gives the double quote to its content

The 'ABBR' tag is used to show the full form of the abbreviations when we hover the cursor over the text
Eg. Hover over the Abbreviation here --> HTML
The full form of HTML appears automatically when we hover over it.

The address tag changes the font to italic.

The BDO tag is used to override the current text direction
Eg. Normal text - This a normal line, read it from left to right
Text after BDO - This a normal line, read it from left to right. The text gets reversed !!
